Orange Site Inspector API

Welcome to the Orange Site Inspector API

This API provides endpoints for managing site inspections, visits, and user authentication.

Available Endpoints:

POST /api/auth/register - Register a new user
POST /api/auth/login - Login with credentials
GET /api/users/profile - Get current user profile
GET /api/sites - Get all sites
POST /api/visits - Create a new visit
GET /api/admin/visits/pending - Get pending visits (Admin only)
POST /api/fileupload/visit-image - Upload visit images
